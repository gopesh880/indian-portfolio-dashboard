
# Indian Portfolio & Risk Intelligence Dashboard

A modern finance analytics dashboard built using Python and Streamlit for Indian retail investors.

## Features

- Portfolio analytics
- Risk scoring
- SIP projections
- Inflation-adjusted wealth analysis
- Monte Carlo simulation
- Portfolio health scoring
- Investor insights
- Goal planning concepts
- Benchmark comparison ideas

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- yfinance
