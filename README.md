
# Indian Portfolio & Risk Intelligence Dashboard

A modern finance analytics dashboard built using Python and Streamlit for Indian retail investors.

## Features

- Portfolio analytics
- Risk scoring
- SIP projections
- Inflation-adjusted wealth analysis
- Monte Carlo simulation
- Portfolio health scoring
- Investor insights
- Goal planning concepts
- Benchmark comparison ideas

## Tech Stack

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- yfinance

## How To Run

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Start the app

```bash
streamlit run app.py
```



This project focuses on realistic portfolio analytics instead of fake AI stock prediction.

It demonstrates:
- risk analysis
- financial planning
- data visualization
- probabilistic thinking
- dashboard engineering
